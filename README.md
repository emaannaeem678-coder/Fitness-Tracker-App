My Fitness App is a simple fitness tracking application developed using HTML, CSS, and c++ as backend. The application allows users to:

Enter personal fitness data
Log daily activities
Calculate calories burned
View fitness achievements

This project was originally developed in C++ and later converted into a web-based application.
eatures
1. Data Entry

Users can enter:

Weight (kg)
Height (cm)
2. Activity Logging

Users can log the duration of:

Walking
Running
Cycling
Jogging
Skipping
3. Calories Calculator

The application calculates calories burned based on activity duration.

4. Achievement Section

Displays:

User weight
User height
Total calories burned
